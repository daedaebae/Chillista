## ðŸš€ Project Overview

**BaristaSim** is a chill, coffee shop simulator game built with **React** and **Vite**.

- **Tech Stack**: React, Vanilla CSS (Modules), Vite.
- **Aesthetic**: 8-bit Pixel Art, Cozy/Parchment UI.
- **Core Loop**: Brew coffee \u2192 Serve customers \u2192 Earn money \u2192 Upgrade shop.
