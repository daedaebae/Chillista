# Jules Features

Features to be implemented by Jules.

## Core Features

- [ ] **User Authentication**
    - Implement a system for users to sign up and log in.
    - Cloud save synchronization (optional but recommended with auth).

- [ ] **Save Game Slots**
    - Allow players to have multiple save files (e.g., Slot 1, Slot 2, Slot 3).
    - UI to select, load, overwrite, and delete save slots.
    - Display save details (Day, Funds, Playtime) on the slot selection screen.

- [ ] **Subtle/Ambient Coffeeshop Environment Sounds**
    - Add background audio layers for coffee shop ambience.
    - Examples: Low chatter, distant traffic, cup clinking, espresso machine hissing.
    - Ensure sounds loop seamlessly and have independent volume control from Music/SFX.
