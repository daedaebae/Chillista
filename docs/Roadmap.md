# Roadmap

This document outlines the development roadmap for Chillista, aligned with our current priorities.

## Phase 2.5: Mobile Focus
**Goal**: Optimize the existing web experience for mobile devices immediately.
- [ ] **Mobile Support**: Ensure basic functionality on iOS/Android browsers.
- [ ] **Touch Controls**: Optimize clickable areas and interactions.
- [ ] **UI Polish**: Verify layout scaling and responsiveness.
- [ ] **Testing**: Validate on real devices (iOS/Android).

## Phase 3: React Native Migration
**Goal**: Transition from a web-app to a native mobile app.
- [ ] **Migration Plan**: Audit codebase for RN compatibility.
- [ ] **Branch Setup**: Create `react-native` branch.
- [ ] **Feature Mapping**: Port features one by one:
    -   Game Loop / Time
    -   Inventory / Save System
    -   UI / Components
- [ ] **Build & Deploy**: Setup fastlane/Xcode/Android Studio workflows.

## Phase 3.5: Reputation System
**Goal**: Introduce progression mechanics beyond just cash.
- [ ] **Core System**: Track `Reputation` alongside `Cash`.
- [ ] **Milestones**: Levels 0-10, unlocking features at each stage.
- [ ] **Effects**: High rep attracted better customers (Tips/Patience).
- [ ] **Purchase Confirmation**: UI popups for high-stakes decisions.

## Phase 4: Mechanics & Balance
**Goal**: Deepen the gameplay loop.
- [x] **Cart Decoration**: Skins (Wood, Marble, Metal) - *Completed*
- [ ] **Difficulty Settings**: Toggle Cozy / Challenging / Extreme. (Partially implemented)
- [ ] **Patience Mechanics**: Visual timers (Green/Yellow/Red). (Partially implemented)
- [ ] **Complex Controls**: Multi-step Aeropress (Boil -> Grind -> Pour -> Press).
- [ ] **Upgrades Expansion**:
    -   Unlock categories (Workstation vs Cart).
    -   Upgrade Levels (e.g. Grinder Lvl 1 -> Lvl 2).

## Phase 5: Management & Economy
**Goal**: Endgame content.
- [ ] **Staff**: Hire baristas to automate clicking.
- [ ] **Events**: Rush hours, Festivals, Weather events.
- [ ] **Recipes**: Latte art minigame, seasonal drinks.
- [ ] **Statistics**: Lifetime stats, best day tracking.

## Wishlist / Backlog
-   **Cloud Save**: Sync progress across devices.
-   **Social**: Share your cart design.
